## Getting Started

This project is a starting point for a Flutter application.

### Google Sign-In Setup

To use Google Sheets integration, you need to set up Google Sign-In:

1. Follow the instructions in [GOOGLE_SIGN_IN_SETUP.md](GOOGLE_SIGN_IN_SETUP.md)
2. Configure SHA-1 fingerprint
3. Set up OAuth client on Google Cloud Console
4. Enable Google Sheets API
